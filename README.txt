SafePulse Full MVP Bundle

Contains:
- lib/main.dart  (full Flutter MVP code)
- pubspec.yaml   (Flutter dependencies)
- PRIVACY_POLICY.txt  (draft)
- TERMS_OF_USE.txt    (draft)
- PLAY_STORE_DESCRIPTION.txt (draft)
- icons/  (Play Store-ready icon sizes)

How to run:
1. Ensure Flutter SDK is installed.
2. In this folder, run:
   flutter pub get
   flutter run

Your developer can integrate these icons into the Android/iOS build and prepare the signed release for Google Play.
